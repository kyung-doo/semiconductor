he purpose of this compatibility testing program is to review the compliance with the SAMSUNG's module and many different motherboard manufacturers using a variety of chipsets. These lists are not intended to be a comprehensive collection of motherboards that will reliably operate with Samsung's module and this compatibility testing program is not intended to replace the system vendor's qualification process.

These lists do not guarantee all compatibility. They are informational only and represent only result of specific test. Compatibility results do not address operational environment system architecture or other variables that must be addressed in product design. This page publication service does not review or address systems that have not tested as being compatible with Samsung memory.
