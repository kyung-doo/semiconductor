Heighten your computing experience

PC users are choosing high-speed SSDs for better computing performance and reliability. The Samsung 750 EVO SSD is the ideal choice for high-performance everyday computing. With sequential read and write speeds up to 540/520 MB/s and random read and write speeds up to 97,000/88,000 IOPS*, your multitasking will be faster than ever. * The sequential write speed was measured using TurboWrite mode. The random read and write speeds were measured using the 250GB capacity model in 4K QD32.

Enjoy fast computing for everyday use

With TurboWrite technology and RAPID Mode, the 750 EVO offers you a fast computing experience. TurboWrite technology accelerates sequential write speeds by creating a write buffer inside the SSD. Magician software provides the option to enhance PC performance with RAPID Mode, which processes data on a system level using unused PC memory (DRAM) as a cache.

Secure data with data encryption

The 750 EVO features encryption technology to help keep all your data safe. An AES-256-bit hardware-based encryption engine secures your data without the performance degradation.

Depend on solid reliability

The flash-based 750 EVO provides superior resistance against shock and vibration for computing on the go. It also includes a warranty for 3 years or up to 70 Terabytes Written (TBW).* *3 years or TBW (120 GB: 35 TBW; 250 GB: 70 TBW), whichever comes first

Discover easy installation

The 750 EVO comes with software to help entry-level users easily install and manage their SSD. Samsung Data Migration software provides a powerful custom-cloning feature that ensures migration works even when the source data is greater than the SSD capacity. And Magician software simplifies SSD management, enabling even novices to maximize the performance and lifetime of their SSD.
