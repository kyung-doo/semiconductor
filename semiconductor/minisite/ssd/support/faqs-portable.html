PC

  • The software for setting the password on the T3 does not run.

    The T3 Security Enabler software for using T3’s password protection feature requires a certain version of OS or higher. Please see below for supported OS versions.

    • Windows OS: Windows 7 or higher
    • Mac OS: Mac OS X 10.7 or higher
  • The LED indicator on the T3 remains on when my Windows PC is in power save mode or turned off.

    It is normal that the LED indicator turns off when your Windows PC enters power save mode or turns off. But it may stay on, depending on USB charging settings in your PC’s BIOS.

  • I am using Windows OS, and I cannot copy files to the T3.

    If the T3 loses power connection or if the T3 is unplugged in an unsafe way, Windows OS may prevent files from being copied to the T3. This is Windows OS’s restriction on the exFAT file system. It does not mean that the product has a problem. To resolve the issue, please follow the below steps.

    • Click the “Scan and Fix” notification that appears when you connect the T3 to your PC.
    • If you closed the notification window, select and right-click the T3 icon on the system tray (notification area) and use the ‘Run CHKDSK’ command.
    • If the T3 icon isn’t displayed on the system tray, open Explorer and select the T3 drive → Right-click → Properties → Tools → click Check.
  • You cannot use security functions without installing the SAT driver in Mac OS X

    The SAT (SCSI ATA Translation) driver is required to use the security functions of the T3 in Mac OS X. Please install the SAT driver in order to use the security features of Samsung Portable SSD and change its settings.

ANDROID MOBILE DEVICE

  • The “USB device was removed unexpectedly” message appears on the Notification View of the Android device to which my T3 is connected.

    When the Android app for T3, Samsung Portable SSD, is running, the T3 app takes over control of the USB interface to achieve T3’s high level of security. And that is why the message appears, but you can check that he Android device and T3 maintain USB connectivity though the app.

    ※ If the app isn’t running, the message will only appear when the T3 is disconnected.

  • If you set your password for the T3 while you are connected to an Android mobile device, you may not be able to log in from a PC if the password contains a smile Emoji or other emoticons, or if the password was copy & pasted instead of being typed manually.

    Smile Emojis and other emoticons will not be recognized as part of the password due to discrepancies between the keyboards of the Android OS and PC OS. Please use only numbers and/or letters.

  • Audio files stop playing when I connect T3 to my mobile device.

    Music may stop playing because the mobile file manager prioritizes T3’s operation first. Playback will resume after T3’s operation is completed.

  • If I disconnect the OTG cable after checking that file copying is completed in the status bar of the mobile device, I cannot open the copied files.

    It is recommended that when files are copied, you use the Safely Remove Device function before removing and reconnecting the OTG cable. Please do not remove the OTG cable right away.

  • The mobile app doesn’t recognize two or more T3 units.

    The T3 app only supports one device.

  • My mobile device’s OS freezes when I connect it to T3 while using certain security programs.

    There may be compatibility issues with certain security programs for mobile devices. Samsung does not guarantee compatibility with such security programs. If the OS of your mobile device freezes, reboot the mobile device or contact the device’s manufacturer. (If you use KNOX, please update it to the latest version.)

  • Files stored in T3 do not run.

    The T3 Log In Activator software for T3 requires a certain version of Android OS or higher. Please see below for supported OS versions.

    · Android: KitKat (ver. 4.4) or higher

OTHER DEVICE

  • T3 isn’t recognized by non-PC devices.

    T3 was developed only for PC (Windows OS and Mac OS) and mobile devices. When connected to other devices, T3 may not be recognized or its functions may be restricted depending on the devices. Also, if password protection is enabled on T3, you cannot access the data stored in T3 because you cannot enter the password on other devices. Please disable password protection when you connect T3 to non-PC or Android devices.

    (e.g.) Many of the latest TVs support photo, music and video playback from external storage via USB. However, not all TVs support this function. When connecting T3 to a TV, please check the following.

    • After connecting T3 to the TV’s USB port, check if T3 is recognized in the TV’s menu.

      If it isn’t recognized, please refer to the TV’s user manual or contact the manufacturer to check for USB support. If it is recognized, see if you can play videos, music, and photos stored in T3 without an issue.

    • If you cannot play them, check the file system of the external storage supported by the TV. T3 is formatted using exFAT file system by default.
      If the TV only supports other file systems, format T3 accordingly.

      * Formatting will delete all data. Please back up your data first.

      If the file system matches, but you cannot play videos, music, and photos, check the TV’s supported file extensions. Playback may not be supported depending on the file extensions supported by the TV. For more details about other devices, please refer to their user manual or contact the manufacturer.

  • Can I use T3 for OS’s other than Windows, Mac OS and Android OS?

    You can only use T3 on PCs running Windows 7 or later or Mac OS X 10.7 or later. You can use T3 on other operating systems and devices as long as password protection is disabled. But there may be restrictions depending on the devices.

SOFTWARE

  • I can’t remember my password. Can I recover my password?

    You cannot. If you forget your password, you cannot remove or update it. Please make sure not to forget your password.

  • If I forget my password, how can I use T3?

    When you forgot the password, you need to reset the T3 to factory settings via an online service by the Samsung Service Center. When you reset it to factory settings, all data stored in it will be deleted. So it is recommended to back up important data prior to reset. For more details, please contact the Service Center indicated on your User Manual.

  • Can I change my password?

    You can change your password on the T3 Settings screen. Tap “>” on the corresponding tab of the settings you wish to change. You can access the Settings screen from the system tray icon on Windows OS, from the Desktop on Mac OS, and from the Samsung Portable SSD app on Android mobile devices.

  • I’m currently not using a password. How do I set one?

    You can enable password protection on the Settings screen of T3. Tap “>” on the corresponding tab of the Settings screen to change your settings. You can access the Settings screen from the system tray icon on Windows OS, from the Desktop on Mac OS, and from the Samsung Portable SSD app on Android mobile devices.

    If you cannot find the icon on the system tray or Desktop, you can download and install the software from the Samsung website
    (http://www.samsung.com/samsungssd).

  • I want to disable password protection.

    You can disable password protection on the Settings screen of T3. Tap “>” on the corresponding tab of the Settings screen to change your settings. You can access the Settings screen from the system tray icon on Windows OS, from the Desktop on Mac OS, and from the Samsung Portable SSD app on Android mobile devices.

  • The password input screen (“Login page”) does not appear.

    If the login page does not appear automatically, select the “T3_SETUP” drive recognized by the system and run “T3 Log In Activator for Win.exe / T3 Log In Activator for Mac.app” to bring up the login page.

OTHERS

  • T3 isn’t recognized by my PC.

    Please make sure that it is connected securely to the USB port.

    The USB port you are using may have an issue. Try connecting it to another USB port. If T3 is connected securely to a USB port, but isn’t recognized by the system, contact the Service Center indicated on your User Manual.

  • The capacity of T3 is displayed as 128 MB.

    For high level of security, T3 is divided security area and user data area. If password protection is enabled and T3 isn’t activated yet, only the security partition containing T3 Log In Activator Software, which is 128 MB, is supposed to be displayed.

    Once you run “T3 Log In Activator for Win.exe / T3 Log In Activator for Mac.app” in the T3_SETUP drive and complete activation, the actual size of the product will be displayed.

    For more details about activation, please refer to “Activation” in the User Manual.

  • My new T3 already has files stored in it.

    The T3 Security Enabling software for password protection is stored in T3 by default. For instructions about how to use it, please refer to “Activation” on your User Manual.

  • T3’s capacity is recognized as smaller than its actual capacity.

    The capacity displayed on your PC may vary from the capacity marked on the product, depending on the drive’s format, partitioning and computer system.
    * Capacity marked on the product is calculated as below.

    • 1 MB = 1,000,000 bytes
    • 1 GB = 1,000,000,000 bytes
    • 1 TB = 1,000,000,000,000 bytes
  • T3’s data transfer is slow.

    T3 supports the max speed of 450 MB/s in the Windows 7 PC environment with USB 3.0 and UASP Mode support. Speed may vary depending on the user’s PC environment and the type of data transferred.

    If it is connected to USB 2.0 or if UASP Mode is not supported, T3 may operate at a lower speed.

  • What is UASP?

    UASP stands for USB Attached SCSI Protocol, and it supports standard SCSI commands along with USB protocol, in order to increase data transfer speed.
    For the UASP function, please download a program related to USB 3.0 from the website of your PC or motherboard’s manufacturer.
    T3 is formatted using exFAT, to support both Windows and Mac OS at the same time.
    To use T3 on devices that only support NTFS, please reformat the drive using NTFS.
    Please refer to the manual for more details about the file system.
