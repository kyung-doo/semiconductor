  • Samsung では顧客の代わりに SSD の取り付けを行っていますか?

    いいえ。Samsung は、いかなる種類の顧客向けの SSD 取り付けサービスも提供していません。 ユーザー マニュアルの取り付けガイドを参照してください。
    SSD の取り付けがうまくいかない場合は、お近くの PC ストアや独立系サービス センターに支援を依頼してください。

  • デスクトップ PC 取り付けキット、ラップトップ PC 取り付けキット、基本パッケージの違いは何ですか?

    各パッケージには同じ SSD が含まれていますが、異なるアクセサリが含まれています。
    以下の各パッケージ デザインのアクセサリ リストを参照してください。

    • デスクトップ: 取り付けガイド、保証概要、USB-SATA ケーブル、SATA データ ケーブル、ネジ、ブラケット、CD (Data Migration ソフトウェア、Magician ソフトウェア、保証書、One-Stop Install Navigator)
    • ラップトップ: 取り付けガイド、保証概要、USB 3.0-SATA ケーブル、ドライブ スペーサ、CD (Data Migration ソフトウェア、Magician ソフトウェア、保証書、One-Stop Install Navigator)
    • 基本: 取り付けガイド、保証概要、CD (Data Migration ソフトウェア、 Magician ソフトウェア、保証書、One-Stop Install Navigator)
  • Samsung SSD では、SATA 3Gb/s (SATA II) または SATA 1.5Gb/s (SATA I) がサポートされていますか?

    はい。 Samsung SSD は、SATA 3Gb/s (SATA II) および SATA 1.5Gb/s (SATA I) との下位互換性があります。 ただし、SATA 3Gb/s (SATA II) および SATA 1.5Gb/s (SATA I) での使用はお勧めしません。 最大限のパフォーマンスを得るために、マザーボードのアップグレードを検討することをお勧めします。

  • Samsung SSD はどのオペレーティング システムでサポートされていますか?

    すべてのオペレーティング システムで SSD がサポートされていますが、一部のオペレーティング システムは、SSD のパフォーマンスを最大限に保つよう最適化されていません。 Windows XP や Windows Vista などの古いオペレーティング システムは、TRIM などの SSD のメンテナンス機能の一部をネイティブにサポートしていません。 Windows XP と Windows Vista のユーザーは、Samsung SSD Magician を使用して、SSD のメンテナンスを手動でスケジュールおよび実行できます。 現在のところ、Samsung SSD Magician は、Mac OS X や Linux が動作するコンピューターではサポートされていません。

  • AHCI を設定するにはどうすれば良いですか?

    AHCI を有効にするための最善の方法は、BIOS で AHCI モードを有効にしてから オペレーティング システムをインストールすることです。

    • システムを起動し、BIOS の設定を開始し、SATA の設定を IDE から AHCI に変更します。
    • オペレーティング システムをインストールします。

    BIOS の設定で AHCI モードを有効にせずに Windows をインストールした場合、Windows の起動時にエラーが発生する可能性があります。
    以下に示す設定を変更するか、Windows を再インストールしてください。

    • [スタート] > [ファイル名を指定して実行] を選択し、[ファイル名を指定して実行] ウィンドウで「regedit」と入力し、[Enter] キーを押してレジストリ エディターを起動します。
    • HKEY_LOCAL_MACHINE > SYSTEM > CurrentControlSet > services > msahci を選択します。このメニューに、Start という名前の項目があります。 ダブルクリックして "0"に変更し、再起動します。
    • BIOS の設定を開始し、SATA の設定を IDE から AHCI に変更します。
      各種メインボードで BIOS セットアップを開始する方法:

      ASUS Intel チップセット: [Main] > [Storage Configuration] > [Configure SATA as] このメニューで、[AHCI] を選択します。

      ASUS AMD チップセット: [Main] > [Storage Configuration] > [SATA Mode Select] このメニューで、[AHCI Mode] に変更します。

      GIGABYTE Intel チップセット: [Integrated Peripherals] > [SATA RAID/AHCI Mode] このメニューで、[AHCI] を選択します。

      GIGABYTE AMD チップセット: [Integrated Peripherals] > [OnChip SATA Type] このメニューで、AHCI を選択します。

      MSI Intel チップセット/AMD チップセット: [Integrated Peripherals] > [On-Chip ATA Devices] > [RAID Mode] で、[AHCI] を選択します。

  • SATA 3Gb/s (SATA II) マザーボードがあります。 拡張カードを購入して、SATA 6GB/s (SATA III) のサポートを追加する必要がありますか?

    SATA 6Gb/S をサポートする拡張カード (SATA III) は、ピーク パフォーマンスに達しないことがあります。
    そのような拡張カードを使用することはお勧めしません。 最大限のパフォーマンスを得るために、マザーボードのアップグレードを検討することをお勧めします。

  • ファームウェアとは何ですか? 更新するにはどうすれば良いですか?

    ファームウェアは、NAND チップなどの半導体チップに格納されたソフトウェア プログラムです。
    これには、デバイスを操作するために必要なソフトウェア コードとコマンドが含まれています。
    ファームウェアは、メーカーがパフォーマンスと品質を向上できるように、ソフトウェア アップデートを通じて更新できるようになっています。

    Samsung SSD のファームウェアは、SSD に付属している Samsung Magician ソフトウェアを 通じて更新できます。
    Magician CD をなくした場合は、次の Samsung の Web サイトから Samsung Magician をダウンロードできます。
    www.samsung.com/samsungssd.

  • ファームウェア更新を更新するにはどうすれば良いですか?

    Samsung では、SSD のパフォーマンス、互換性、またはユーザー エクスペリエンスを向上するために、新しいファームウェアが提供されることがあります。 新しいファームウェアがリリースされたら、ファームウェア更新を行うことをお勧めします。
    以前のモデルの SSD (470 シリーズ) を使用していて、ライブ ファームウェア更新機能で問題が発生した場合は、起動可能な CD/DVD/USB を介した、DOS ベースのファームウェア更新ユーティリティを使用することができます。

    • 1.Magician を使用したファームウェア更新: www.samsung.com/samsungssd

      選択した Samsung SSD でファームウェア更新が推奨されている場合、[新しいバージョンのファームウェアが検出されました (バージョン番号)。] というメッセージが、[ファームウェア更新] 画面の [更新] ボタンの横に表示されます。(注) 新しいファームウェアが利用可能な場合、画面の右下隅に通知が表示されます。.

    • 2. DOS ファームウェア更新ユーティリティ: www.samsung.com/samsungssd
    注意)
    ファームウェアを更新する前に、データをバックアップし、実行中のすべてのアプリケーション (Magician を除く) を終了してください。
    ファームウェアを更新する前に、ご使用の SSD のファームウェアが最新版かどうかを確認してください。

    Magician のホーム画面で -> [ファームウェア更新] を選択

    Samsung の Web サイト: www.samsung.com/samsungssd

    ファームウェアの更新中はコンピューターの電源を切らないでください。
    ファームウェア更新は、RAPID が有効な SSD では実行できません。
    PC の再起動が必要です。
  • ファームウェアが利用可能な場合、更新は必要ですか?

    新しいバージョンが利用可能になったら、最新のファームウェア バージョンにアップグレードすることをお勧めします。
    SSD のファームウェアが古い場合は、できるだけ最新のファームウェア バージョンに更新してください。

  • SSD のファームウェアを更新するにはどうすれば良いですか?

    SSD のファームウェアを更新するための最も簡単な方法は、Samsung Magician ソフトウェアを使用することです。
    このソフトウェアは Windows のみで使用でき、Linux や Mac OS X ではサポートされていません。
    次の Web サイトで入手可能なファームウェア ISO ファイルを使用して、起動可能な USB ドライブや起動可能な CD/DVD を作成することもできます。
    www.samsung.com/samsungssd.

  • DOS モードでファームウェアを更新するにはどうすれば良いですか?

    注意: 以下の手順は、必要なすべてのファイルが USB/CD/DVD ディスク上に存在していることを前提としています。
    詳細については、「起動可能な USB の作成」または「起動可能な CD/DVD の書き込み」セクションを参照してください。

    • 1. BIOS で、USB/CD/DVD ドライブをプライマリ起動デバイスとして構成します。
      注: システムで AHCI/IDE/ATA モードを有効にしてください。
    • 2. 起動可能な USB/CD/DVD を使用して、コンピューターを DOS モードで再起動します。
    • 3. ファームウェア更新ユーティリティが自動的に起動します。
    • 4. 免責事項を読んだら、[Y] を押して条項に同意します。
    • 5. アプリケーションによって、接続されている SSD がスキャンされます。 検出されたディスクのリストが 表示されます。
    • 6. [Y] を押して確認し、ファームウェア更新プロセスを続行します。
      注意: ファームウェア更新が完了するまで、システムを操作しないようにしてください。
    • 7. ディスクが最新のファームウェアに更新されます。
    注意)
    ファームウェアを更新する前に、各自のデータをバックアップし、
    ファームウェアを更新する前に、ご使用の SSD のファームウェアが最新版かどうかを確認してください。

    Magician のホーム画面で -> [ファームウェア更新] を選択

    Samsung の Web サイト: www.samsung.com/samsungssd

    ファームウェアの更新中は電源を切らないでください。
    ファームウェア更新は、RAPID が有効な SSD では実行できません。
    SSD はメインボードの SATA ポート 0 に接続してください。
    PC のシャットダウンが必要です。
  • 起動可能な USB ディスクを作成するにはどうすれば良いですか?
    • 1. USB フラッシュ ドライブを、コンピューターの利用可能な USB ポートに挿入します。 コンピューターでデバイスが認識されるまで待ちます。
    • 2. Magician を実行し、Magician の [ファームウェア更新] メニューの [参照] ボタンをクリックします。
    • 3. 接続されている USB フラッシュ ディスクを選択します。
    • 4. [開始] ボタンをクリックします。 進行状況バーに、起動可能な USB ディスク作成プロセスのステータスが表示されます。
    • 5. 正常に完了したら、コンピューターを再起動し、USB ディスクから起動して、ファームウェア更新プロセスを開始します。
    • 6. 「DOS ファームウェア ユーティリティ」セクションの手順に従います。
  • 起動可能な CD/DVD ディスクを作成するにはどうすれば良いですか?
    • 1. CD/DVD を光学式ドライブに挿入します。 システムでデバイスが認識されるまで待ちます。
    • 2. Magician の [ファームウェア更新] メニューの [参照] ボタンをクリックします。
    • 3. 接続されている CD/DVD ドライブを選択します。
    • 4. [開始] ボタンをクリックします。 進行状況バーに、起動可能な CD/DVD 作成プロセスのステータスが示されます。
    • 5. 正常に完了したら、コンピューターを再起動し、CD/DVD から起動して、ファームウェア更新プロセスを開始します。
    • 6. 「DOS ファームウェア ユーティリティ」セクションの手順に従います。
  • Linux コンピューターまたは Macintosh でファームウェアを更新するにはどうすれば良いですか?

    x86 ベースの Linux マシンを使用している場合は、ファームウェア ISO ファイルを起動可能な USB/CD/DVD ドライブに Windows ベースのコンピューターからロードします。 使用している Macintosh で USB/CD/DVD からの起動がサポートされている場合も、この方法でファームウェアをアップグレードできます。 ファームウェアの ISO ファイルは www.samsung.com/samsungssd からダウンロードできます。

  • SSD のファームウェアをアップグレードするための他の方法はありますか?

    Magician ソフトウェアを実行し、ナビゲーション メニューから [ファームウェア更新] を選択します。
    ここから、使用している SSD に最新のファームウェアがインストールされているかどうかを確認できます。
    Samsung Magician ソフトウェアは Windows のみで利用でき、Linux と Mac OS X ではサポートされていません。

  • 使用している Samsung SSD のファームウェアが最新かどうかはどうすればわかりますか?

    Magician ソフトウェアを実行し、ナビゲーション メニューから [ファームウェア更新] を選択します。
    ここから、使用している SSD に最新のファームウェアがインストールされているかどうかを確認できます。
    Samsung Magician ソフトウェアは Windows のみで利用でき、Linux と Mac OS X ではサポートされていません。

  • 新しいファームウェアを既存の SSD にインストールすると、ドライブに格納されているデータは消去されますか?

    最新のファームウェアを SSD にインストールしても、ドライブに格納されているデータは消去されませんが、潜在的なリスクを軽減するために、ファームウェアを更新する前にデータを別のドライブにバックアップすることを強くお勧めします。

  • ファームウェアを最新版から古いバージョンにダウングレードすることはできますか?

    いいえ。ファームウェアをアップグレードしたら、古いバージョンに戻すことはできません。
    パフォーマンス、信頼性、互換性が最適化されていることを確認するため、すべてのファームウェア リリースは当社により十分にテストされているため、古いファームウェア バージョンに戻す必要はありません。

  • USB2SATA ケーブルを使用してファームウェアを更新できますか?

    現在、Magician は USB - SATA インタフェースをサポートしていません。
    デスクトップ PC の SATA ケーブルにディスクを接続するか、ラップトップ PC の内蔵ドライブ ベイにディスクを取り付けて、更新してください。

  • Samsung SSD には特殊なデバイス ドライバーが必要ですか?

    いいえ。SSD は、標準的な HDD と簡単に交換できます。

  • Samsung SSD 用のデバイス ドライバーはどこにありますか?

    Samsung SSD 用のデバイス ドライバーはありません。
    すべての Samsung SSD は、プラグ アンド プレイ デバイスとして動作するので、デバイスのケーブルを接続すればすぐにシステムで認識されます

  • Samsung SSD には特殊な SATA 6Gb/s (SATA III) ケーブルが必要ですか?

    いいえ。 すべての SATA ケーブルは、SATA 6Gb/s (SATA III) と互換性があります。
    一部のベンダーは特殊なケーブルを販売していますが、最高のパフォーマンスを得るうえで必要ではありません。

  • SSD をコンピューターに取り付けるにはどうすれば良いですか?

    ユーザー マニュアルの取り付けガイドか、One-stop Install Navigator を参照してください。 www.samsung.com/samsungssd

  • SSD を起動ドライブとして取り付けるにはどうすれば良いですか?

    SSD を新しいシステムに取り付ける場合には、オペレーティング システムをインストールするときにコンピューターに SSD のみを接続することで、SSD を確実に起動ドライブにすることができます。

  • 2.5 インチ SSD をデスクトップ コンピューターに取り付けるにはどうすれば良いですか?

    使用している 3.5 インチ ハード ディスク ドライブを、デスクトップ コンピューターのシャーシ内部の電源ケーブルと SATA インタフェース コネクターから抜きます。
    両方のケーブルを 2.5 インチ SSD に接続します。 2.5 インチ SSD を、シャーシ内の上下に動いたり破損したりしない場所に静かに置きます。
    構造上の完全性を高めるため、3.5 インチアダプター ブラケットを 2.5 インチ SSD に取り付けることができます。 これらのブラケットは、デスクトップ取り付けキットに付属しています。
    3.5 インチ ブラケットがない場合は、オンラインまたは各地の IT ストアで購入できます。

  • 最善の RAID 構成は何ですか?

    最善の RAID 構成は何ですか?
    「最善の」 RAID 構成はありません。
    RAID 構成を選ぶユーザーは、各自の利用シナリオに最も適した方法を選択する必要があります。
    RAID 構成には、それぞれに利点と欠点があります。
    できるだけパフォーマンスを高めることを望んでいるユーザーは、2 台以上の SSD を RAID 0 モードで構成することができます。
