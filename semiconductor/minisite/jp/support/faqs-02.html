  • SSD に問題が起きたらどうすれば良いですか?

    取り付けがうまくいかない場合は、取り付けガイドを参照してください。
    製品の使用上の問題がある場合は、FAQ を参照するか、既知の問題に対する解決策を参照してください。
    製品を返却する必要がある場合は、SSD を購入した販売店に連絡するか、S-ASC (Samsung 認定サービス センター) にお問い合わせください。

  • Windows Vista または Windows 7 から起動しようとしたときに、"bootmgr is missing error" を解決するにはどうすれば良いですか?

    Windows 回復ディスクを作成し、それを実行して "bootmgr is missing error" を解決します。OS の修復ディスクの作成方法については、Microsoft または使用しているコンピューターのメーカーにお問い合わせください。
    bootmgr は、Windows のインストール ディスクで修復オプションを選択することでも修復できます。
    詳細は、OS のヘルプかサポート技術情報を参照してください。
    OS を修復できない場合は、ディスクをフォーマットして OS を再インストールしてください。

  • BIOS で SSD が検出されない場合はどうすれば良いですか?
    • 1.システムの BIOS が古い場合、SSD がサポートされていない可能性があります。 最新版の BIOS に更新してください。
    • 2.SSD ケーブルが適切に接続されていないと、検出に失敗することがあります。 ケーブルを抜き、再度接続してください。
    • 3.SATA ポートに問題があると、BIOS が検出に失敗する可能性があります。 新しい SATA ポートに移動し再度試してください。
  • デバイス マネージャーに SSD が表示されない場合、どうすれば良いですか?
    • 1.システムで SSD が自動的に認識されない場合があります。Windows デバイス マネージャーで新しいデバイスを探してください。
    • 2.システムの BIOS の設定で、各ポートのホット プラグが既定でオフになっていると、Windows が SSD の検出に失敗することがあります。
      システムを再起動し、BIOS のセットアップを開始し、[SATA Configuration] メニューを選択して、各ポートのホット プラグが [Enable] になっていることを確認してください。
      各種メインボードで BIOS セットアップを開始する方法 :

      GIGABYTE -->F12, ASUS --> F8, ASRock --> F11, Biostar --> F9, MSI --> F11, Intel --> F10

    • 3.SSD ケーブルが適切に接続されていないと、SSD の検出に失敗することがあります。 ケーブルを抜き、再度接続してください。
    • 4.パーティションが定義されていないと、SSD の検出に失敗することがあります。
      デバイス マネージャーでパーティションを設定し、ドライブをフォーマットしてから、再起動してください。

      [管理] > [ディスクの管理] > SSD を選択 > パーティションの設定とフォーマット

    LSI MegaRAID カードを使用している場合:
    LSI MegaRAID カードを使用している場合、SSD がデバイス マネージャーに表示されるようにするには、ディスクを論理ボリュームとして構成する必要があります。 ディスクは、MegaRAID の BIOS 設定画面で論理ボリュームとして構成する必要があります。この画面は、再起動後のブート シーケンスで表示されます。

  • ドライブ選択に SSD が表示されない場合、どうすれば良いですか?
    • 1.SATA ケーブルを交換し、再度試します。
    • 2.ボード上の SATA コネクターにゴミが付着していたり、損傷していないことを確認します。
    • 3.SSD ケーブルが適切に接続されていないと、SSD の検出に失敗することがあります。 ケーブルを抜き、再度接続してください。
    • 4.SSD を別の SATA ポートに移動し、ドライブが検出されるかどうかを確認します。

      SSD が拡張ポートに接続されているかどうかを確認します。 その場合は、SSD を通常のポートに接続します。

      SSD が通常のポートに接続されている場合は、別のポートに移動して再起動します。

    • 5.SSD が適切に接続されているものの、ドライブ選択に表示されない場合は、以下の手順を試します。

      [マイ コンピューター] アイコンを右クリックし、[管理] を選択して、[記憶域] を選択します。[ディスクの管理] をクリックし、ディスクが設定されていることを確認します。

      ディスクが有効になっており、ダイナミック ディスクとして設定されていることを確認します。そうなっていない場合は、その設定をダイナミック ディスクに変更します。

    • 6.SSD を別のコンピューターに接続し、SSD 自体に問題があるかどうかを確認します。

    LSI MegaRAID カードを使用している場合:
    LSI MegaRAID カードを使用している場合、SSD がデバイス マネージャーに表示されるようにするには、ディスクを論理ボリュームとして構成する必要があります。
    ディスクは、MegaRAID の BIOS 設定画面で論理ボリュームとして構成する必要があります。この画面は、再起動後のブート シーケンスで表示されます。

  • Windows 7 Professional で、Samsung SSD が SSD としてではなく通常の ATA ドライブとして検出されます。 これは正常ですか?

    はい。Windows 7 で SSD が通常の ATA ドライブとして検出されるのは正常です。 Windows 7 では、SSD の設定が最適化されることに注意してください。

  • OS をインストールできない場合はどうすれば良いですか?
    • 1.OS のインストール CD に損傷がないことを確認します (傷、指紋、汚れなど)。
    • 2.SATA ケーブルを交換して再起動します。
    • 3.ボード上の SATA コネクターにゴミが付着していたり、損傷していないことを確認します。
    • 4.SSD ケーブルが適切に接続されていないと、SSD の検出に失敗することがあります。
      ケーブルを抜き、再度接続してください。
    • 5.SSD を別の SATA ポートに移動し、ドライブが検出されるかどうかを確認します。

      SSD が拡張ポートに接続されているかどうかを確認します。 その場合は、SSD を通常のポートに接続します。

      SSD が通常のポートに接続されている場合は、別のポートに移動して再起動します。

    • 6.Windows 7 以降では、システム領域が既存の HDD にインストールされる可能性があります。そのため、OS をインストールする前に SSD のみを接続してください。
    • 7.SSD を別のコンピューターに接続し、SSD 自体に問題があるかどうかを確認します。

    (注) 「使用している OS の CD が正規品でない場合、OS インストール時に SSD が認識されないことがあります。」

    サーバー システムの SAS ポートに接続された SATA SSD に OS をインストールする場合:
    SSD が接続されている SAS チップセット コントローラーが OS で認識されない場合、SSD が認識されないことがあります。
    たとえば、Windows OS をインストールする場合、C602 Patsburg (CPU) に組み込まれた SCU (Storage Control Unit) に接続された SSD は表示されないことがあります。
    この場合、OS のインストール時に SCU 関連のデバイス ドライバーをロードして SSD が認識されるようにしてから、SSD に OS をインストールする必要があります。

  • ドライブをフォーマットできない場合はどうすれば良いですか?
    • 1.SSD が適切に接続されていないと、検出されないことがあります。 ドライブを接続し直し、再度試してください。
    • 2.ボード上の SATA コネクターにゴミが付着していたり、損傷していないことを確認します。
    • 3.SATA ケーブルを交換し、再度試します。
    • 4.SSD を別の SATA ポートに移動し、ドライブが検出されるかどうかを確認します。

      SSD が拡張ポートに接続されているかどうかを確認します。 その場合は、SSD を通常のポートに接続します。

      SSD が通常のポートに接続されている場合は、別のポートに移動して再起動します。

    • 5.BIOS の設定を確認します (必ず SATA モードの設定を確認してください)。
    • 6.6SSD を別のコンピューターに接続し、SSD 自体に問題があるかどうかを確認します。
    • 7.Secure Erase を実行して SSD を初期化し、再度試します。 (Secure Erase の FAQ を参照)

    (注) 「使用している OS の CD が正規品でない場合、OS インストール時に SSD が認識されないことがあります。」
    サーバー システムの SAS ポートに接続された SATA SSD に OS をインストールする場合:

  • SSD の速度が遅いように感じる場合は、どうすれば良いですか?
    • 1.メインボードで SATA Gen3 がサポートされていることを確認してください。
      SATA Gen2 メインボードでは速度が制限され、SSD の動作が通常よりも遅くなることがあります。
    • 2.以下に示すパフォーマンス関連のすべての設定が正しいことを確認します。

      書き込みキャッシュが有効になっている必要があります (FAQ を参照)

      AHCI が有効になっている必要があります (FAQ を参照)

      ドライバーのバージョンを確認し、必要に応じて更新します (FAQ を参照)

  • SSD のパフォーマンスが時間とともに低下するのはなぜですか?

    一般に、SSD テクノロジは、他のあらゆる種類のストレージ デバイスよりも高い安定性と持続的なパフォーマンスを提供します。
    しかし、デバイスにフルにデータが格納されるか、予測された P/E サイクル回数を超えると、初期の空の状態よりもパフォーマンスが低下することがあります。
    SSD を長期間使用すると、SSD 内部でハウスキーピング処理を行う必要が生じるため、SSD の速度が低下することが あります。
    この問題を解決するには、Samsung Magician ツールを使用し、[パフォーマンス最適化] を定期的に実行してください。
    問題が続く場合は、Samsung SSD Magician ソフトウェアを通じて Secure Erase を実行できます。このプロセスの詳細な情報と手順については、Magician のマニュアルを参照してください。 Magician ソフトウェア アプリケーションは、Window ベースの PC のみで利用できます。
    ドライブのオーバー プロビジョニング領域と SSD で必要なハウスキーピング処理の数によっては、適切なレベルのバッファー領域を維持するために SSD で実行される内部処理が、ドライブのパフォーマンスに影響を与えることがあります。 その場合、オーバー プロビジョニング領域を増やすことでドライブのパフォーマンスが改善することがあります。
    最高のパフォーマンスを維持するために、SSD Magician を使用して SSD コントローラー用のオーバー プロビジョニング領域を確保するか、SSD 上に最低 10% の空き領域を意図的に確保しておくことをお勧めします。

  • SSD Magician で SSD が検出されないのはなぜですか? どうすれば良いですか?

    SSD Magician は、Samsung ブランドの SSD のみと互換性があります。
    ご使用の SSD が OEM 製品の場合、Magician ではサポートされません。
    Samsung ブランドの SSD を使用している場合は、SSD が Windows のデバイス マネージャーで検出されているかどうかを確認してください。
    SSD は、IDE デバイスまたは AHCI デバイスとして検出されます。
    検出されない場合は、Microsoft Windows 用の標準の AHCI コントローラー ドライバーをインストールすることで問題が解決される可能性があります。

  • Magician をインストールした後で SSD が検出されない場合はどうすれば良いですか?
    • 1.SSD は、ドライバーの互換性の問題が原因で検出されない可能性があります。ドライバーを最新版に更新してください。
    • 2.古い AMD 製チップセットと AMD ドライバーの一部は、Magician でサポートされていません。
      これに該当する場合は、MS の既定のドライバーをインストールしてください。
    • 3.SSD が RAID/SCSI ストレージ デバイス コントローラーに接続されている場合、プログラムが SSD を適切に検出できないことがあります。
      これに該当する場合は、MS の既定のドライバーをインストールしてください。
  • Magician で状態ステータスまたは TBW が正しく表示されない場合はどうすれば良いですか?
    • 1.Samsung 470 シリーズ SSD は、ドライブ ステータス表示機能 (状態ステータスと TBW) をサポートしていません。
    • 2.ドライブ ステータスは SMART 値を使用するため、すべての SMART 値が正しく表示されることを確認するには、Magician のホーム画面で [SMART] ボタンを押します。
    • 3.ドライバーの互換性の問題により、ホストから SSD へのコマンドが失敗する場合は、SMART 値が表示されず、ドライブ ステータスが表示されません。
      CrystalDisk Info などの別のプログラムを使用してください。
    • 4.デバイス マネージャーを起動し、SSD に関連付けられているドライバーを選択し、右クリックして [プロパティ] - [ドライバー] を選択し、[ドライバーの更新] を実行します。
    • 5.IRST ドライバーを最新版に更新してください。
  • パフォーマンス最適化エラーが発生した場合は何をすれば良いですか?

    Windows XP/2003 で SSD の容量が 137GB 未満の場合、ATA PASS THROUGH 機能の不具合により問題が発生する可能性があります。
    Microsoft より提供されているホット フィックスをダウンロードしてインストールしてください。
    ダウンロード リンク: http://support.microsoft.com/kb/934205.

    上記の解決策で問題が解決されない場合は、サービス センターにお問い合わせください。

  • Magician を使用して OS 最適化を実行しているときにエラーが発生した場合は、何をすれば良いですか?
    • 1.Magician が OS のサービスを最適化しようとして失敗する場合は、手動によるサービスのオン/オフを試みてください。
    • 2.OS 最適化では、OS のサービスがオン/オフされますが、状況によっては OS で許可されない可能性があります。パーソナル コンピューターのメーカーにお問い合わせください。
    • 3.IRST がインストールされている場合、Magician を使用して「書き込みキャッシュ バッファー」と「書き込みキャッシュ バッファーのフラッシュ」を有効/無効にしようとすると、エラーが発生することがあります。
      この機能を使用するには、IRST を手動で削除した後で Magician を使用してやり直すか、Windows のデバイス マネージャーで機能を有効/無効にする必要があります。
    • 4.仮想メモリの設定を変更した場合は、システムを再起動してください。
    • 5.Magician を使用して電源オプションのモードを変更することはできません。 電源オプションのモードを変更するには、[コントロール パネル] の [電源オプション] を使用してください。
    • 6.一部のメーカーの PC では、システムの復元オプションのモードが固定されている場合があります。 その場合、Magician を使用してシステムの復元オプションのモードを変更することはできません。 この値の変更については、PC メーカーにお問い合わせください。
    • 7.3 つの OS 最適化モード (パフォーマンスの最適化/領域の最大化/信頼性の向上) のうち 1 つを選択し、いずれかの OS 最適化項目を手動で変更した場合、OS 最適化モードは自動的に [ユーザー定義] に切り替わります。
    • 8.OS 最適化項目を Magician 以外の方法 (コントロール パネルなど) により手動で調整した場合、その変更は Magician 更新時には反映されません。
    • 9.3 つの OS 最適化モードのいずれかを選択した後でラップトップの画面が暗くなった場合は、[コントロール パネル] > [電源オプション] > [Samsung High Performance] で設定を変更できます。
  • [OS 最適化] を実行した後で電源管理の設定が変更された場合は何をすれば良いですか?

    [OS 最適化] を実行した後で電源管理の設定を変更する場合は、[コントロール パネル] - [電源オプション] で設定を変更してください。

  • Magician で [ドライブ ステータス] が [失敗] と表示される場合はどうすれば良いですか?
    • 1.SMART 値を確認してください。 SMART 値がない場合は、デバイス マネージャーで SATA コントローラー ドライバーを更新するか、MS インボックス ドライバー (Standard AHCI Serial ATA Controller) に切り替えます。
    • 2.SMART 値が「SMART FAIL」の場合、Samsung サービス センターにお問い合わせください。
  • Samsung Data Migration ソフトウェアを使用するときに何に注意すべきですか?

    ソース ドライブ (HDD など) の内容を複製する場合、ターゲット ディスク (Samsung SSD) 上のすべてのデータは削除され、復元できなくなります。 そのため、引き続き保持するデータをターゲット ディスクに保存している場合は、事前にそのデータをバックアップしてください。
