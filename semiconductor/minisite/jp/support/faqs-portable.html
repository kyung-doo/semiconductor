PC

  • T3 にパスワードを設定するためのソフトウェアが起動しません。

    T3 のパスワード保護機能を使用するための T3 Security Enabler ソフトウェアには、特定バージョン以降の OS が必要です。サポートされている OS バージョンについては、以下を参照してください。

    • Windows OS: Windows 7 以降
    • Mac OS: Mac OS X 10.7 以降
  • Windows PC が省電力モードまたは電源がオフの状態だと T3 の LED インジケーターが点灯したままになります。

    Windows PC が省電力モードまたは電源がオフの状態になると、通常は LED インジケーターが点灯しなくなります。ただし、PC の BIOS に登録されている USB の充電設定によっては、点灯したままになる場合があります。

  • Windows OS を使用していますが、ファイルを T3 にコピーできません。

    T3 の電源接続が失われた場合、または安全でない方法で T3のケーブルを外した場合、Windows OS では T3 にファイルをコピーできない可能性があります。これは Windows OS の exFAT ファイル システム上の制限によるものです。製品に問題があるわけではありません。この問題を解決するには、以下の手順に従ってください。

    • T3 を PC に接続したときに表示される [スキャンおよび修復] 通知をクリックします。
    • 通知ウィンドウを閉じた場合は、システム トレイ (通知領域) の T3 アイコンを右クリックして、[CHKDSK の実行] コマンドを選択します。
    • システム トレイに T3 アイコンが表示されない場合は、エクスプローラーを開いて T3 ドライブを選択し、右クリックして [プロパティ] → [ツール] の順に選択して、[チェック] をクリックします。
  • Mac OS X で SAT ドライバーをインストールしないとセキュリティ機能を使用することができません。

    Mac OS X で T3 のセキュリティ機能を使用するには、SAT (SCSI ATA Translation) ドライバーが必要です。Samsung Portable SSD のセキュリティ機能を使用するには、SAT ドライバーをインストールして、設定を変更してください。

ANDROID モバイル デバイス

  • T3 が接続されている Android デバイスの通知ビューに「USB デバイスが予期せず取り外されました」というメッセージが表示されます。

    Samsung Portable SSD の T3 用 Android アプリを実行すると、T3 の高レベルのセキュリティを維持するために、T3 アプリが USB インターフェイスの制御を引き継ぎます。このようなメッセージが表示されるのはそのためですが、アプリによって Android デバイスと T3 で USB 接続が維持されていることを確認できます。

    ※ アプリを実行していない場合、このメッセージは T3 の接続が切断されたときのみ表示されます。

  • Android モバイル デバイスに接続中に T3 のパスワードを設定する場合、パスワードに絵文字のスマイルやその他の顔文字が含まれていたり、パスワードを入力する代わりにコピー アンド ペーストしたりすると、PC からのログインができません。

    Android OS と PC OS のキーボード間の不一致により、絵文字のスマイルやその他の顔文字は、パスワードの一部として認識されません。パスワードには英数字のみを使用してください。

  • T3 をモバイル デバイスに接続すると、オーディオ ファイルの再生が停止します。

    モバイル ファイル マネージャーでは T3 の処理が最優先されるため、音楽の再生が停止する場合があります。T3 の処理が完了すると、音楽の再生は再開します。

  • モバイル デバイスのステータス バーでファイルのコピーが完了したことを確認してから OTG ケーブルを取り外しましたが、コピーされたファイルを開くことができません。

    ファイルをコピーするときは、OTG ケーブルを取り外して再接続する前に、デバイスを安全に取り外す機能を使用することをお勧めします。OTG ケーブルはすぐに取り外さないでください。

  • モバイル アプリで 2 台以上の T3 が認識されません。

    T3 アプリがサポートするのは 1 台のデバイスのみです。

  • 特定のセキュリティ プログラムを使用中に T3 に接続すると、モバイル デバイスの OS がフリーズします。

    モバイル デバイス用の特定のセキュリティ プログラムに互換性の問題があると考えられます。そのようなセキュリティ プログラムとの互換性は保証されません。モバイル デバイスの OS がフリーズした場合は、モバイル デバイスを再起動するか、デバイスの製造元に問い合わせてください(KNOX を使用している場合は、最新バージョンに更新してください)。

  • T3 に保存したファイルを実行できません。

    T3 用の T3 Log In Activator ソフトウェアには、特定バージョン以降の Android OS が必要です。サポートされている OS バージョンについては、以下を参照してください。

    • Android: KitKat (バージョン4.4) 以降

その他のデバイス

  • PC 以外のデバイスで T3 が認識されません。

    T3 は PC (Windows OS および Mac OS) とモバイル デバイス向けのみに開発されたものです。その他のデバイスに接続した場合、T3 は認識されないか、デバイスによっては機能が制限されることがあります。また、T3 でパスワード保護が有効な場合、その他のデバイスではパスワードを入力できないため、T3 に保存されているデータにアクセスすることはできません。T3 を PC 以外または Android 以外のデバイスに接続する場合は、パスワード保護を無効にしてください。

    たとえば、最新型テレビの多くは、USB 経由での外付けストレージからの写真、音楽、ビデオの再生に対応していますが、すべてのテレビがこのような機能に対応しているわけではありません。T3 をテレビに接続する場合は、以下のことを確認してください。

    • T3 をテレビの USB ポートに接続し、テレビのメニューで T3 が認識されるかどうかを確認します。

      認識されない場合は、テレビのユーザー マニュアルを参照するか、製造元に問い合わせて USB サポートを確認してください。

    • 認識された場合は、T3 に保存されているビデオ、音楽、写真を問題なく再生できるかどうかを確認します。

      再生できない場合は、テレビが対応している外付けストレージのファイル システムを確認します。T3 はデフォルトでは exFAT ファイル システムでフォーマットされています。テレビがその他のファイル システムのみに対応している場合は、そのファイル システムで T3 をフォーマットします。


      * フォーマットすると、すべてのデータは削除されます。データをバックアップしてからフォーマットしてください。

      ファイル システムが一致してもビデオ、音楽、写真の再生ができない場合は、テレビが対応しているファイルの拡張子を確認します。テレビが対応しているファイル拡張子によっては、再生に対応していない場合があります。
      その他のデバイスの詳細については、ユーザー マニュアルを参照するか、製造元に問い合わせてください。

  • Windows、Mac OS、Android OS 以外の OS で T3 を使用することはできますか?

    T3 は Windows 7 以降または Mac OS X 10.7 以降を実行している PC でのみ使用できます。パスワード保護が無効になっていれば、その他のオペレーティング システムおよびデバイスでも T3 を使用できます。ただし、デバイスによっては制限がある場合があります。

ソフトウ

  • パスワードを忘れてしまいましたが、パスワードを解除することはできますか?

    パスワードを忘れた場合、パスワードの解除、削除または更新はできません。パスワードは忘れないようにしてください。

  • パスワードを忘れてしまった場合、どうすれば T3 を使用することができますか?

    パスワードを忘れてしまった場合は、Samsung サービス センターのオンライン サービスを利用して T3 を工場出荷状態にリセットすることができます。但し、工場出荷時の設定にリセットすると、保存されていたデータはすべて削除されますので、重要なデータは常日頃からバックアップを取っておくことをお勧めします。詳細については、ユーザー マニュアルに記載されているサービス センターに問い合わせてください。

  • パスワードを変更することはできますか?

    パスワードは T3 の [設定] 画面で変更できます。変更したい設定のタブで [>] をタップします。[設定] 画面は Windows OS のシステム トレイ アイコン、Mac OS のデスクトップ、Android モバイル デバイスの Samsung Portable SSD アプリからアクセスできます。

  • 現在、パスワードを使用していません。パスワードを設定するにはどうすればいいですか?

    T3 の [設定] 画面でパスワード保護を有効にします。[設定] 画面のタブで [>] をタップして、設定を変更します。[設定] 画面は Windows OS のシステム トレイ アイコン、Mac OS のデスクトップ、Android モバイル デバイスの Samsung Portable SSD アプリからアクセスできます。

    システム トレイまたはデスクトップでアイコンが見つからない場合は、Samsung の Web サイト (http://www.samsung.com/samsungssd) からソフトウェアをダウンロードしてインストールしてください。

  • パスワード保護を無効にしたいのですが。

    T3 の [設定] 画面でパスワード保護を無効にすることができます。[設定] 画面のタブで [>] をタップして、設定を変更します。[設定] 画面は Windows OS のシステム トレイ アイコン、Mac OS のデスクトップ、Android モバイル デバイスの Samsung Portable SSD アプリからアクセスできます。

  • パスワードの入力画面 (ログイン ページ) が表示されません。

    ログイン ページが自動的に表示されない場合は、システムによって認識されている T3_SETUP ドライブを選択して、T3 Log In Activator for Win.exe または T3 Log In Activator for Mac.app を実行すると、ログイン ページが表示されます。

一般

  • T3 が PC で認識されません。

    T3 が USB ポートにしっかりと接続されていることを確認してください。

    使用している USB ポートに問題がある場合があります。別の USB ポートに接続してみてください。T3 が USB ポートにしっかりと接続されているのにシステムによって認識されない場合は、ユーザー マニュアルに記載されているサービス センターに問い合わせてください。

  • T3 の容量が 128 MB と表示されます。

    セキュリティを強化するために、T3 ではセキュリティ領域とユーザー データ領域に分かれています。パスワード保護が有効になっていても、T3 のアクティベーションが完了していない場合、T3 Log In Activator ソフトウェアが含まれている 128 MB のセキュリティ パーティションしか表示されません。

    T3_SETUP ドライブで T3 Log In Activator for Win.exe または T3 Log In Activator for Mac.app を実行して、アクティベーションが完了すると、製品の実際のサイズが表示されるようになります。

    有効化の詳細については、ユーザー マニュアルの「アクティベーション」を参照してください。

  • 新しい T3 にファイルがすでに保存されています。

    デフォルトではパスワード保護用の T3 Security Enabling ソフトウェアが T3 に保存されています。使用方法については、ユーザー マニュアルの「アクティベーション」を参照してください。

  • T3 の容量が実際の容量より少なく認識されます。

    ドライブのフォーマット、パーティション分割、コンピューター システムによっては、PC に表示される容量が製品に記載されている容量と異なる場合があります。
    * 製品に記載されている容量は、以下のように計算されます。

    • 1 MB = 1,000,000 バイト
    • 1 GB = 1,000,000,000 バイト
    • 1 TB = 1,000,000,000,000 バイト
  • T3 のデータ転送が遅いです。

    USB 3.0 および UASP モードに対応した Windows 7 PC 環境で T3 がサポートする最大速度は 450 MB/秒です。転送速度はユーザーの PC 環境や転送するデータのタイプによって異なる場合があります。

    USB 2.0 に接続している場合、または UASP モードがサポートされていない場合は、T3 の転送速度が遅くなることがあります。

  • UASP とは何ですか?

    UASP は USB Attached SCSI Protocol の略語で、データ転送速度を上げるために、標準の SCSI コマンドを USB プロトコルでサポートします。
    UASP 機能を使用するには、PC またはマザーボードの製造元の Web サイトから USB 3.0 の関連プログラムをダウンロードしてください。
    Windows と Mac OS の両方を同時にサポートするために、T3 は exFAT でフォーマットされています。NTFS にのみ対応しているデバイスで T3 を使用する場合は、NTFS でドライブを再フォーマットしてください。ファイル システムの詳細については、付属のマニュアルを参照してください。
