  • Magician とは何ですか?

    Samsung Magician ソフトウェアは、ユーザーが Samsung SSD の状態およびパフォーマンスを簡単に管理できるように設計されています。
    Magician は重要な SSD 状態ステータス情報を提供するほか、ユーザーがファームウェアの更新、パフォーマンスの測定、Windows オペレーティング システムの構成の最適化を行うのをサポートします。
    Samsung Magician ソフトウェアは、Samsung Solid State Drive (SSD) の所有者専用に開発および配布されています。

    最小仕様および推奨されるドライバーなど、Magician の詳細は、www.samsung.com/samsungssd にある Magician のインストール ガイドを参照してください。
    www.samsung.com/samsungssd

  • Samsung Magician ソフトウェアを使用すると何ができますか?

    Samsung Magician では、認証、パフォーマンス ベンチマーク、パフォーマンス最適化、ファームウェア更新、OS の最適化、オーバー プロビジョニング、Secure Erase、RAPID mode、データ セキュリティの使用方法ガイドの機能が提供されています。
    また、SSD のパフォーマンスと寿命を最大化するためのオペレーティング システムの最適化にも役立ちます。

  • Samsung SSD Magician ソフトウェアは Linux または Mac OS X をサポートしていますか?

    現時点で、Samsung SSD Magician ソフトウェアは Windows ベースの PC のみで動作します。

  • Magician ではすべての SSD がサポートされていますか?

    SSD Magician は、Samsung ブランドの SSD のみをサポートしています。 Samsung 製の OEM SSD はサポートされていません。

  • Samsung SSD Magician ソフトウェアは、多言語版の Windows OS をサポートしていますか?

    Samsung SSD Magician アプリケーションは、以下の 10 の言語のみをサポートしています。
    - 英語、韓国語、スペイン語、ドイツ語、フランス語、イタリア語、ロシア語、ポルトガル語、中国語 (簡体字)、日本語

  • Magician をインストールできない場合はどうすれば良いですか?
    • 1.Magician をインストールできない場合はどうすれば良いですか?

      [Uninstall Samsung Magician] を選択するか、コントロール パネルの [プログラムと機能] で [プログラムのアンインストールまたは変更]、[アンインストール] の順に選択します。

    • 2.以前のバージョンの Magician がインストールされていたフォルダーを削除します。

      既定のパス: C:\Program Files\Samsung Magician or C:\Program Files\Samsung\Samsung Magician

    • 3.以前のバージョンの Magician のレジストリを削除します。

      [ファイル名を指定して実行] で「regedit」と入力し、 [コンピューター] の下の HKEY_LOCAL_MACHINE\SOFTWARE\Samsung Magician を削除

    • 4.Magician を再インストールします。
  • Magician には何らかの制限がありますか?

    Magician を使用する前に、以下の情報をしっかりとお読みください。

    • 1.Magician は Samsung ブランドの SSD 用に特別に設計されています。 ファームウェア更新、ディスク状態ステータス、Secure Erase、Total Bytes Written などの SSD 固有の機能は、Samsung SSD 以外ではサポートされていません。
    • 2.Magician は SCSI コントローラー インタフェースを介して接続されている SSD では動作しません。
    • 3.MBR および GPT パーティション タイプのみがサポートされています。 Magician は他のパーティション タイプでは動作しない可能性があります。
    • 4.Magician はユーザー パスワードでロックされている SSD では動作しません。
    • 5.ユーザーは Magician の [システム情報] でディスクのリストを手動で更新し、接続されているまたは取り外されたすべてのディスクを正確に反映する必要がある場合があります。
    • 6.ベンチマーク結果は、キューの深さ 32 に対するものです。
    • 7.新たに接続されたディスクのベンチマークまたはパフォーマンス最適化を実行する前に、[システム情報] でディスクのリストを手動で更新する必要がある場合があります。
    • 8.パフォーマンス最適化機能では ATA PASS THROUGH 機能を使用して TRIM コマンドが SSD に渡されます。 SSD のサイズが 137 GB 未満である場合、この機能は Windows XP/2003 によって無効にされている場合があります。 Microsoft によって、この問題に関するホット フィックスが http://support.microsoft.com/kb/934205で提供されています。
    • 9.Secure Erase は Windows 8 および 8.1 システムではサポートされていません。
    • 10.パフォーマンス最適化では NTFS ファイル システムのみがサポートされています。
    • 11.システムで不適当なクラスターが存在する場合、十分な空き領域が存在してもオーバー プロビジョニングが失敗する場合があります。
    • 12.ベンチマークは一部のリムーバブル ストレージ デバイスで動作しない場合があります。
    • 13.ユーザーが OS 管理ツールのディスク管理を使用してボリューム サイズを拡張または縮小できない場合、動的オーバープロビジョニングが適切に機能しない場合があります。
    • 14.システムで IRST ドライバーが有効になっているか、インストールされている場合、 SSD での書き込みキャッシュ バッファーまたはフラッシュの有効化に失敗する場合があります (OS 最適化)。
      * また、特定または最新の IRST ドライバー (IRST 12.6.0.1033、12.8.10.1005) がインストールされている場合、認証機能が適切に動作しない可能性があります。 この場合、古いバージョンにロールバックしてください。
    • 15.SATA 構成の RAID on mode は Samsung Magician および DOS アプリケーションではサポートされていません。
    • 16.PC はファームウェア更新後に自動的にシャットダウンします (Magician でシャットダウン前に 20 秒のカウントダウンが行われます)。
    • 17.セキュリティ モード (Class 0、TCG/Opal、または Encrypted Drive) は、削除前または他の PC へのインストール前に無効にする (ロック解除する) 必要があります。
    • 18.AMD コントローラーに接続された Samsung ブランドの SSD では、ファームウェア更新が失敗する場合があります。 既定の sata ahci コントローラー (Microsoft ドライバー) を使用してやり直してください。
    • 19.ドライバーによってマルチ スレッド操作 (IO) が処理されない場合、パフォーマンス ベンチマークが Asmedia コントローラーでタイムアウトになる可能性があります。
    • 20.必ず最新のストレージ ドライバーを使用してください。 カスタム ストレージ ドライバーを使用している場合、Magician が適切に動作しない可能性があります。
    • 21.Magician は、Magician のインストール前に調整された電源オプションのみを保持します。 インストール後に電源オプションが変更された場合、Magician はその設定を保持できません。

    詳細は、Magician のインストール ガイドを参照してください。
    - www.samsung.com/samsungssd

  • SSD の状態ステータスを確認するにはどうすれば良いですか?

    Samsung Magician ツールを実行することで Samsung SSD の状態ステータスを確認できます。
    また、SSD の S.M.A.R.T 属性を確認できる、独立したソフトウェア プログラムを実行することもできます。

  • 製品が正規品であることを確認するにはどうすれば良いですか?

    購入した Samsung SSD 製品が正規品であることを確認するには、CD に収録されている Magician ソフトウェアを使用します。
    Magician のホーム画面の [ディスク ドライブ] ウィンドウで、[シリアル番号] の横の [認証] ボタンをクリックします。
    製品が正規品である場合は、[正規] と表示されます。 そうでない場合は [非正規] と表示されます。

  • Magician を使用してファームウェアを更新するにはどうすれば良いですか?

    1. Samsung SSD をバックアップします。
    新しいファームウェアのインストール中は SSD を取り外さないでください。
    予測できない結果が発生する可能性があります。
    ファームウェアのインストール プロセスが完了するのを待ってから、他の操作を開始してください。
    2. Magician を除く、開いているすべてのアプリケーションを閉じます。
    3. [新しいバージョンのファームウェアが検出されました (バージョン番号)。] というメッセージが、[ファームウェア更新] 画面の [更新] ボタンの横に表示されます。
    [更新] をクリックして、選択した Samsung SSD のファームウェアを更新します。
    この操作中は、システムの使用を最小限に抑えるようにしてください。
    4. 正常に完了したら、[ファームウェア更新が完了しました。] というメッセージが表示されます。

    • 注意)
      ファームウェアを更新する前に、ご使用の SSD のファームウェアが最新版かどうかを確認してください。
    • ファームウェアを更新する前に、データをバックアップし、実行中のすべてのアプリケーション (Magician を除く) を終了してください。

      Magician のホーム画面で -> [ファームウェア更新] を選択

      Samsung の Web サイト : www.samsung.com/samsungssd

    • ファームウェアの更新中は電源を切らないでください。
    • ファームウェア更新は、RAPID が有効な SSD では実行できません。
    • SSD はメインボードの SATA ポート 0 に接続してください。
    • PC のシャットダウンが必要です。
  • Samsung Magician で DOS モードで Secure Erase を実行しようとしたときに、ドライブが「凍結状態」であると表示されます。 どうすれば良いですか?

    Secure Erase 中に「凍結状態」メッセージに遭遇した場合は、PC の電源を停止せずに、SATA 電源ケーブルを SSD から抜き、数秒待ってから再度接続します。
    コマンド プロンプトで「SEGUI0.exe」と入力し、Magician DOS ユーティリティを再度実行します。 Secure Erase を再度実行します。

  • Samsung Data Migration ソフトウェアは何をするものですか?

    Samsung Data Migration ソフトウェアは、ユーザーが現在のオペレーティング システム、アプリケーション ソフトウェア、およびユーザー データを含むすべてのデータを、既存のストレージ デバイス (HDD など) から新しい Samsung SSD に迅速、簡単、かつ安全に移行できるように設計されています。

    詳細については、Samsung の Web サイトにあるユーザー ガイドを参照してください。
    - www.samsung.com/samsungssd

  • Samsung Data Migration には何らかの制限がありますか?

    Samsung Data Migration ソフトウェアの制限は、すべてインストール ガイドに記載されています。
    インストール ガイドは、 www.samsung.com/samsungssdからダウンロードできます。

    • 1.このソフトウェアは、ユーザー マニュアルに記載されている Windows オペレーティング システムのみをサポートしています。
    • 2.このソフトウェアは、ユーザー マニュアルに記載されている Samsung SSD のみをサポートしています。
      コンピューター メーカーまたはその他のチャネルから提供された OEM ストレージ デバイスはサポートされていません。
    • 3.このソフトウェアは、オペレーティング システムがソース ディスクにインストールされている場合にのみ実行できます。
      オペレーティング システムのインストール環境がないディスク上では機能しません。
    • 4.ソース ディスクに保存されているデータ量がターゲット ディスクの容量を超えている場合、複製は行われません。
      ソース ディスク ("C:" ドライブなど) には、20% 以上の空き領域がある必要があります。
      ソース ディスクの 80% を超えてデータが保存されている場合、複製プロセスは正常に行われません。
      セカンダリ パーティションには、5 GB 以上の空き領域が必要です。
    • 5.2 つ以上のドライブ ("C:"、"D:"、および "E:" ドライブなど) があり、"C:" ドライブにオペレーティング システムがインストールされているシステムでは、最初の 2 つのドライブのみが複製されます。
      Windows のインストール時に作成されたシステム パーティションは、自動的に複製されます。
    • 6. コンピューター メーカーで出荷前に生成された OEM リカバリ パーティションは複製できません。
      ただし、Samsung 製の PC に SRS (Samsung Recovery Solution) 5 または 6 がインストールされている場合は, リカバリ パーティションが自動的に複製されます (SRS 5 より前のバージョンはサポートされていません)。
    • 7. 複製後にソース ディスクとターゲット ディスクのデータ容量に数 GB の差が生じるのは正常です。
      このソフトウェアでは、Windows によって仮想メモリを管理するために自動的に生成されたファイル (ページ ファイルおよび休止状態ファイルなど) はコピーされません。
    • 8. このソフトウェアでは、暗号化されたディスクは複製できません。
      暗号化されたディスクを複製するには、最初にそのディスクのパスワードを解除する必要があります。
    • 9. 複製時にメインボード チップセット ドライバーが最新でない場合、このソフトウェアは正しく機能しない場合があります。
    • 10. PC に複数のオペレーティング システムがインストールされている場合 (例: Windows 7 が C ドライブにインストールされ、Windows 8 が D ドライブにインストールされている)、複製されたディスクは正しく機能しない場合があります。
    • 11. ソース ディスクが損傷している場合 (例: ディスクに不良セクターがある)、複製されたディスクは正しく機能しない場合があります。
    • 12. このソフトウェアを使用してディスクを複製するときは、複製前に他のプログラムをすべて閉じて、十分なメモリを解放することをお勧めします。
    • 13. システムにインスタント PC リカバリ ソフトウェアがインストールされている場合、このソフトウェアは正しく機能しない場合があります。
    • 14. ソース ディスクをダイナミック ディスクに変換している場合、このソフトウェアは正しく機能しない場合があります。
  • ソース ディスクの容量がターゲット ディスクより大きい場合にディスクを複製するにはどうすれば良いですか?

    動画、音楽、写真などのメディア データを他のストレージに移動するか、できるだけ多くのデータを削除してください。
    データ移行を開始するには、ソース ディスクの容量がターゲット ディスクより小さい必要があります。
    Samsung Data Migration ソフトウェアは、ユーザーの利便性向上のために上記の機能を提供しています。
    ただし、メディア ファイルを移動するには十分なストレージを準備していただく必要があります。

  • ユーザーが利用できる Samsung SSD のツールには何がありますか?

    Samsung は、ユーザー向けに Magician/Migration ツールを提供しています。
    各ソフトウェア ツールには、インストール ガイドと詳細な操作マニュアルが付属しており、システム要件などの重要な情報が記載されています。
    製品を使用する前にガイドをお読みください。

    以下に示すプログラムは、Samsung.com の Web サイトでダウンロードできます。

    1. Samsung Data Migration ソフトウェア
    - Samsung Data Migration ソフトウェアを使用すると、コンピューターの現在のストレージ デバイスに格納されているデータを、新しい Samsung Electronics 製のSSD (Solid State Disk) に、すばやく、簡単に、安全にコピーできます。

    2. Samsung Magician ソフトウェア
    - Samsung Magician ソフトウェアは、ユーザーが Samsung SSD の状態およびパフォーマンスを簡単に管理できるように設計されています。
    Magician は重要な SSD 状態ステータス情報を提供するほか、ユーザーがファームウェアの更新、パフォーマンスの測定、Windows オペレーティング システムの構成の最適化を行うのをサポートします。 Samsung Magician ソフトウェアは、Samsung Solid State Drive (SSD) の所有者専用に開発および配布されています。
